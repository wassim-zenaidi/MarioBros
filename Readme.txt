Titre : Bowser
Type : jeu de plateforme
Créateurs : Zenaidi Wassim

Installaion du jeu :
-Il faut utiliser Algoscript : http://algoscript.info
-Ouvrir le fichier 'Bowser' et copier le code
-Dans l'onglet 'code JS' collez le code
-Aller dans 'file manager' -> 'External FS'
-Cliquer sur 'choisir un fichier' et selectionner le dossier 'images' puis une fois une image ouverte cliquer sur 'data file to Internal FS'
-Répéter l'opération pour toutes les images
-Faire la même chose avec le dossier 'sons'
-Revenir sur l'onglet 'Graphic output' et cliquer sur 'Executer' en haut à gauche

Mode d'emploi : 
pour vous déplacer il vous faut utiliser les touches :
z : pour sauter
s : pour s'arrêter
q : pour aller à gauche (il avance automatiquement)
d : pour aller à droite (il avance automatiquement)
le but est de prendre l'épée et d'aller tuer bowser et d'éviter les flammes
Il y a une certaine probabilité pour que la seconde flamme vous suive 
En appuyant sur la touche 'a' vous êtes en mode invincible
